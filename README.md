## 問題点
・集英社が著作権を持つゴールデンカムイの作品名やキャラクターを使用した不特定多数への集金

・寄付やクラファンサイトではなく自家通販サービスサイトpictSPACEを利用

・企画内容を度々削除変更してその履歴や告知がない

・企画者が寄付に対して控除を受けないことが証明できない
